Hermes Agent Drive

Personal-use automation tool for Google Drive file management

About

Hermes Agent Drive is a personal automation tool that integrates with Google Drive API to perform file management operations as explicitly requested by the user. It runs entirely on your local machine.

Features

  • Google Drive file listing and search
  • File upload and download operations
  • Metadata retrieval and management
  • OAuth 2.0 authentication with local token storage
