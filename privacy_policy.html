Privacy Policy

Last updated: June 8, 2026

Overview

This application ("Hermes Agent Drive Integration") is a personal-use automation tool that integrates Google Drive API for file management purposes. This privacy policy explains what data is accessed and how it is handled.

Data Accessed

  • Google Drive files: The application accesses files in your Google Drive only when explicitly requested by you (the user).
  • File metadata: File names, types, sizes, and modification dates may be read to provide file management functionality.
  • OAuth tokens: Authentication tokens are stored locally on your machine and are never transmitted to third parties.

Data Storage

  • All OAuth credentials and access tokens are stored locally on your computer.
  • No data is sent to external servers except to Google's official API endpoints.
  • No data is shared with any third party.

Data Usage

  • Google Drive files are accessed solely to perform file operations you explicitly request.
  • No automated data collection, analysis, or processing occurs without your direct instruction.

Contact

For questions about this privacy policy, contact the application owner via the email address associated with this Google Cloud project.

Changes

This privacy policy may be updated as needed. Users will be notified of any material changes.
